# FT-048: Distributable Codex Skill

## О разделе

Каталог фиксирует delivery-единицу для repo-local Codex skill package, который
позволяет агентам управлять `zelma` sessions через публичный CLI contract.

## Аннотированный индекс

- [`brief.md`](brief.md)
  Читать, когда нужно понять problem space, scope, non-scope и canonical verify
  contract для skill package.
- [`design.md`](design.md)
  Читать, когда нужно проверить selected solution, C4/system-context boundary,
  feature-local decisions и skill command routing.
- [`implementation-plan.md`](implementation-plan.md)
  Читать, когда нужно выполнить реализацию и верификацию без переопределения
  canonical requirements или design facts.
- [`decision-log.md`](decision-log.md)
  Читать, когда нужно увидеть FPF-решения, закрытые во время review-improve.
